## Gemini CLI
```bash
Any OS (via npm): npm install -g @google/gemini-cli
macOS / Linux (via Homebrew): brew install gemini-cli
macOS (via MacPorts): sudo port install gemini-cli
Kali Linux (via APT): sudo apt install gemini-cli
```
